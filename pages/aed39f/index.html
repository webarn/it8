    最长公共前缀

    # 题目

    编写一个函数来查找字符串数组中的最长公共前缀。
如果不存在公共前缀，返回空字符串 ""。
​
示例 1：
​
输入：strs = ["flower","flow","flight"]
输出："fl"
​
示例 2：
​
输入：strs = ["dog","racecar","car"]
输出：""
解释：输入不存在公共前缀。

    题解分析
    1. 当字符串数组长度为 0 时则公共前缀为空，直接返回;
    2. 取数组的第一个元素作为匹配基准 str, 因为是公共最长,所以后面的一定要和第一个匹配;
    3. 设置一个基准 str 的当前下标值
    4. 遍历基准, 每次多取一个字符串: (第一次: f 第二次: fl 第三次: flo 以此类推)用作比较值: cur;
    5. 遍历后面的元素, 依次和 cur相比较, 如果出现不相等,在说明前面的是相等的,直接返回;
    6. 只有在都相等的情况下 index 才会 ++, 所以 index 的值一直是相等的下标, 所以返回 cur.slice(0, index)

    # 题解

    var strs = ['flower', 'flow', 'flight'];

function long(strs) {
  if (!strs.length) return '';

  let str = strs[0];
  let index = 0;
  let res = 0;

  while (index < str.length) {
    let cur = str.slice(0, index + 1); //
    for (let i = 1; i < strs.length; i++) {
      if (!strs[i].startsWith(cur)) {
        return cur.slice(0, index);
      }
    }
    index++;
  }
}

console.log(long(strs));
