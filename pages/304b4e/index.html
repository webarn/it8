    无重复字符的最长子串

    # 题目

    给定一个字符串 s ，请你找出其中不含有重复字符的 最长子串 的长度。
​
​
示例 1:
​
输入: s = "abcabcbb"
输出: 3
解释: 因为无重复字符的最长子串是 "abc"，所以其长度为 3。
​
示例 2:
​
输入: s = "bbbbb"
输出: 1
解释: 因为无重复字符的最长子串是 "b"，所以其长度为 1。
​
示例 3:
​
输入: s = "pwwkew"
输出: 3
解释: 因为无重复字符的最长子串是 "wke"，所以其长度为 3。
     请注意，你的答案必须是 子串 的长度，"pwke" 是一个子序列，不是子串。
​
示例 4:
​
输入: s = ""
输出: 0

    题解分析
    1. 设置左边边界 l :初始为 0, 设置最大重复数 res :初始为 0, 设置一个 map ,记录出现的位置
    2. 遍历传入的字符串, r 表示右边的边界
    3. 遍历判断当前右边的边界值是否出现在 map 中
    4. 如果出现则把 l 右边界值更新为原位置 + 1

    # 题解

    var s = 'abcabcbb';

function long(str) {
  var l = 0;
  var res = 0;
  var map = new Map();

  for (let r = 0; r < str.length; r++) {
    if (map.has(str[r])) {
      l = map.get(str[r]) + 1;
    }
    res = Math.max(res, r - l + 1);
    map.set(str[r], r);
  }
  console.log(res);
}

long(s);
